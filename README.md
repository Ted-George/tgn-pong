# TGN Pong

Classic Pong built in LÖVE2D (Lua) — CS50 Game Development, Week 0.
Part of **Teganjo Studios**.

## Play it
▶️ [Play on itch.io](https://ted-gaturu.itch.io/pong)

## Features
- Two-paddle local multiplayer
- Ball physics with paddle-angle deflection
- Score tracking and win condition
- Retro pixel-art styling

## Built with
- [LÖVE2D](https://love2d.org/) — Lua game framework
- Delta-time based movement and collision

## Run locally
1. Install [LÖVE2D](https://love2d.org/)
2. Clone this repo
3. Drag the folder onto the LÖVE executable, or run:love .
